## 02 — Variables and Math

Write a function `area_of_rectangle(width, height)` that returns the area.

### Examples

- `area_of_rectangle(3, 4)` -> `12`
- `area_of_rectangle(5, 2)` -> `10`

### Hint

<details>
<summary>Show hint</summary>

1. The area of a rectangle is found by multiplying its width by its height.

2. In Python, multiplication uses the `*` operator.

</details>
