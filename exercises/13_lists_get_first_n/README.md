## 13 — Lists (Get First N)

Write a function `get_first_n(items, n)` that returns a new list containing the first `n` items.

### Examples

- `get_first_n([1, 2, 3, 4], 2)` -> `[1, 2]`
- `get_first_n([1, 2], 5)` -> `[1, 2]`

Assume `n` is a non-negative integer.

<summary>Hint</summary>

Think about how you can take a portion of a list using slicing. What happens if you slice beyond the length of the list?
