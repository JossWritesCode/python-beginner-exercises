## 17 — Dictionaries (Letter Count)

Write a function `letter_count(s)` that returns a dictionary counting each character.

### Example

`letter_count("aba")` -> `{"a": 2, "b": 1}`

<details>
<summary>Hint</summary>

Try looping through each character in the string.  
Use a dictionary to keep track of how many times you've seen each character.

If the character is already in the dictionary, increase its count.  
If not, start its count at 1.

</details>
