## 06 — For Loops (Count Up)

Write a function `count_up(n)` that returns a list of integers from 1 up to `n`.

### Examples

- `count_up(1)` -> `[1]`
- `count_up(5)` -> `[1, 2, 3, 4, 5]`

Assume `n` is a positive integer.

### Hint

<details>
<summary>Show hint</summary>

You’ll want to build a list of numbers one at a time.

Python’s `range()` function is useful for counting in loops.

See the official Python documentation for [`range()`](https://docs.python.org/3/library/stdtypes.html#range).
