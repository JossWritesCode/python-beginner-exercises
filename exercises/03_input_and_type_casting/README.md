## 03 — Type Casting

Write a function `to_int(s)` that converts a string to an integer.

### Examples

- `to_int("7")` -> `7`
- `to_int("0")` -> `0`

### Notes

Assume the string is always a valid integer.

### Hint

<details>
<summary>Show hint</summary>

Python has a built-in function that converts values to integers.

For example:

```python
int("5")
```
